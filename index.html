div is an object that determines the layout of a section of the page. It is a block element and covers all the width from left to right.
span is similar to the div element except that it doesn't cover all the width from left to right if it doesn't need to. Fills only the space needed. Is an in-line element, not a block element. Do not put block elements inside in-line elements.
